# 📖 프로젝트 소개

> `MES`는 MES는 제조업체가 생산 공정을 보다 효과적으로 관리하고 최적화할 수 있도록 돕는 중요한 시스템입니다.</br>
> `Eat-it`는 디저트의 주문, 생산, 재고 관리를 담당하는 종합적인 MES 솔루션으로, 디저트 제조 공정의 효율성을 극대화하고 품질 관리를 향상시키기 위해 설계되었습니다.</br>
